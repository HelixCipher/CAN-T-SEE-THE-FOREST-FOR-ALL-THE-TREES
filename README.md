# CAN'T SEE THE FOREST, FOR ALL THE TREES

### Univariate Analysis and Predictive Modeling

### Overview

This project focuses on univariate analysis of various variables from a dataset, followed by predictive modeling using decision trees and random forests. The aim is to explore and understand different factors affecting student performance.

### Data Exploration Process

#### Dataset Overview:

* Initial exploration of the dataset to check its structure, size, and column types.

#### Univariate Analysis:

* Distribution of Age: Understanding student demographics.

* Study Time Distribution: Analyzing study habits.

* Distribution of Parental Education Levels: Examining the influence of parental education.

### Predictive Modeling

This project employs machine learning techniques to predict student performance using:

* Decision Trees

* Random Forests

* Grid Search for Hyperparameter Tuning
